# Real-Feel Chat Interface 💬

A modern, lightweight, and highly interactive chat application UI built using vanilla web technologies. This project focuses on mimicking the user experience (UX) of professional messaging apps like WhatsApp or Messenger.

## ✨ Features
* **Real-Time Feel:** Includes a "Typing..." indicator and simulated bot responses.
* **Multimedia Support:** Send both text messages and images from your local device.
* **Interactive UI:** Built-in emoji picker for expressive messaging.
* **Message Metadata:** Each message includes dynamic timestamps and "Sent/Read" status updates.
* **Responsive Design:** Fully functional on both desktop and mobile browsers.

CSS3: Custom animations, Flexbox for alignment, and "pop-in" message effects.

JavaScript (ES6): DOM manipulation, FileReader API for images, and simulated interaction logic.
